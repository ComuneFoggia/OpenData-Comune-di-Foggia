# Dataset: Elenco Matrimoni Foggia - Residenti

## Descrizione
Questo dataset contiene i dati relativi ai matrimoni celebrati nel Comune di Foggia nel corso del 2024, riferiti a cittadini **residenti**.  
Il dataset è stato sottoposto a un accurato processo di pre-processing e arricchimento per favorirne la pubblicazione come Open Data.

---

## Struttura del Dataset

| Colonna                             | Descrizione                                                                 |
|-------------------------------------|-----------------------------------------------------------------------------|
| Numero                              | Numero progressivo del matrimonio (ID univoco)                             |
| Data Matrimonio                     | Data della celebrazione (formato YYYY-MM-DD)                               |
| Ora Matrimonio                      | Orario della celebrazione (formato HH:MM)                                  |
| Data Atto                           | Data di registrazione dell’atto                                            |
| Luogo Matrimonio                    | Località o sede della celebrazione                                         |
| Provincia/Stato Estero              | Area geografica di provenienza (provincia italiana o stato estero)         |
| Sigla                               | Sigla della provincia o dello stato estero                                 |
| Anno Celebrazione                   | Anno estratto dalla data del matrimonio                                    |
| Mese Celebrazione                   | Mese (valore numerico) della celebrazione                                  |
| Giorno della Settimana Celebrazione | Giorno della settimana della celebrazione (es. Lunedì, Martedì, ecc.)      |
