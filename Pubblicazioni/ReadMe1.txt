# Dataset: Elenco Pubblicazioni Foggia

## Descrizione
Questo dataset contiene i dati relativi agli atti di pubblicazione registrati dal Comune di Foggia nel corso del 2024.  
Il dataset è stato sottoposto a un accurato processo di pre-processing e arricchimento per favorirne la pubblicazione come Open Data.

---
## Struttura del Dataset

| Colonna                                    | Descrizione                                                                |
|--------------------------------------------|----------------------------------------------------------------------------|
| Numero                                     | Numero progressivo dell'atto (ID univoco)                                  |
| Data Inizio Pubblicazione                  | Data di inizio della pubblicazione (formato AAAA-MM-GG)                    |
| Data Celebrazione                          | Data della celebrazione (formato AAAA-MM-GG)                               |
| Data Fine Pubblicazione                    | Data di fine della pubblicazione (formato AAAA-MM-GG)                      |
| Data Atto                                  | Data di registrazione dell'atto                                            |
| Durata Pubblicazione                       | Numero di giorni tra inizio e fine pubblicazione                           |
| Giorni tra Atto e Inizio Pubblicazione     | Numero di giorni tra la registrazione dell'atto e l'inizio della pubblicazione |

## Periodo di Riferimento
