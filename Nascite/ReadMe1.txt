# Dataset: Elenco Nascite Foggia - Residenti

## Descrizione
Questo dataset contiene i dati relativi agli atti di nascita registrati dal Comune di Foggia nel corso del 2024, riferiti a cittadini **residenti**.  
Il dataset è stato sottoposto a un accurato processo di pre-processing e arricchimento per favorirne la pubblicazione come Open Data.

---

## Struttura del Dataset

| Colonna                 | Descrizione                                                                 |
|-------------------------|-----------------------------------------------------------------------------|
| Numero                  | Numero progressivo dell'atto (ID univoco)                                   |
| Sesso                   | Sesso del bambino (M = Maschio, F = Femmina)                                |
| Data Nascita            | Data di nascita del bambino (formato AAAA-MM-GG)                            |
| Ora Nascita             | Ora di nascita del bambino (formato HH:MM)                                  |
| Comune di Nascita       | Comune di nascita del bambino                                               |
| Sigla Comune di Nascita | Sigla della provincia del comune di nascita                                 |
| Comune di Residenza     | Comune di residenza del bambino                                             |
| Data Atto               | Data di registrazione dell’atto di nascita                                  |
| Giorno Nascita          | Giorno in cui è avvenuto la nascita  (es. 1,2,3...30,31)                    |
| Mese Nascita            | Mese in cui è avvenuta la nascita                                           |
| Fascia Oraria           | Parte della giornata in cui è nato il bambino (es. Mattina, Sera)           |

--
